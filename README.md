# How To Create Tab using HTML & CSS | Vertical Tab Design

Um projeto simples de **tabs verticais** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **menu de abas verticais**, ideal para organizar conteúdo em seções.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Abas verticais
- Layout organizado
- Design simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
